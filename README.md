🚀 Project Overview

Gold price prediction is a critical problem in financial analytics.
This project uses Machine Learning regression techniques to analyze historical gold price data and forecast future trends.

The model captures underlying patterns in historical data and extends them to predict future values.

---

📅 Data & Prediction Range

- 📊 Historical Data: 2014 – 2026
- 🔮 Prediction Range: 2027 – 2032
- 📈 Focus: Trend-based future price estimation

---

🧠 Model Used

- 🌲 Random Forest Regressor
- 📉 Supervised Learning (Regression)

Why Random Forest?

- Handles non-linearity well
- Reduces overfitting
- Provides high accuracy

---

📊 Model Evaluation

The model is evaluated using standard regression metrics:

- MAE (Mean Absolute Error) → Average prediction error
- MSE (Mean Squared Error) → Penalizes large deviations
- R² Score → Measures model accuracy and fit

---

📉 Visualization

- 🔵 Blue Line → Actual Prices (2014–2026)
- 🟠 Orange Line → Predicted Prices (2027–2032)

The graph demonstrates how the model learns historical trends and projects them into the future.
